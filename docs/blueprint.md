# **App Name**: Richie Reference

## Core Features:

- Multi-Pane Dashboard: Organize research papers within a multi-pane dashboard with folders, a searchable list of papers, and a metadata editor.
- Import Papers: Import papers using DOI, BibTeX, or manual PDF upload.
- Custom PDF Viewer: Display PDFs with custom controls for highlighting and sticky notes; store scroll positions.
- Abstract Summarizer: Generate a brief, three-bullet summary of the paper abstract, leveraging AI tool to condense complex information.
- Text Explainer: Provide simplified explanations of highlighted complex text within PDFs, using AI tool to interpret academic jargon.
- Summary coulmn: A column for summary, so a user can view it on the pdf listing page. This is based on the paper's abstract.
- Database integration: Store all the required data to Cloud SQL (PostgreSQL)

## Style Guidelines:

- Primary color: Muted blue (#64748B) for primary actions, aligning with a professional and academic aesthetic.
- Background color: Light gray (#F0F3F5), offering a distraction-free environment with high readability.
- Accent color: Soft teal (#94A3B8), used sparingly for interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective feel.
- Use minimalist icons to represent folders, document actions, and AI functionalities.
- Implement a three-pane layout with a responsive design, ensuring the PDF viewer adapts to different screen sizes.
- Use subtle transitions and animations for interactive elements, like expanding folders or loading summaries.
